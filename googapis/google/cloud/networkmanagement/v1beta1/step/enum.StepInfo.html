Expand description

Configuration or metadata associated with each step. The configuration is filtered based on viewer‚Äôs permission. If a viewer has no permission to view the configuration in this step, for non-final states a special state is populated (VIEWER_PERMISSION_MISSING), and for final state the configuration is cleared.
